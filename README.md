# Lesta Task

## Installation
1. Clone repo - `git@github.com:Chaffe/lesta-task.git`
2. `npm install`

## Usage
**Development**

`npm run dev`

* App served @ `http://localhost:3000`

## Implemented
- SPA application with React, Typescript, Mobx, SCSS, Eslint, Prettier
- Data is stored in `src/api/data.json` and used in request - `src/services/vehicles.service.ts`
- Filtering vehicles list by Premium/Collection tabs
- Filtering vehicles list by vehicle_type
- Sorting vehicles list by price
- Custom scrollbar
- Animated controls buttons and vehicle cards with onHover and onClick animations
- Adaptive UI for all device types
