Пожелания по реализации:
• Используйте React, TypeScript.
• Набор товаров должен быть в формате JSON.
• На вкладках «Премиальная» и «Коллекционная» должны быть разные наборы товаров.
• Сделать переключение между вкладками «Премиальная» и «Коллекционная» с помощью роутинга.
• Реализуйте возможность фильтрации карточек товаров с танками по классам техники: одновременно
можно выбрать несколько классов техники. Если не выбран ни один класс, то показывается вся
техника.
• Реализуйте возможность сортировки карточек товаров по актуальной цене (по возрастанию или
убыванию).
• Иконка на кнопке сортировки должна переворачиваться в зависимости от выбранного направления
сортировки.
• С анимациями любых элементов можно импровизировать на своё усмотрение. Хотя бы раз нужно
использовать анимированный эффект.
• Стилизация бокового скролла будет плюсом, но не обязательна.
• Приложение должно работать в актуальной версии Google Chrome.
• Адаптивная верстка: перестроение лейаута на небольших разрешениях на своё усмотрение (важно,
чтобы страница оставалась функциональной для пользователя).
Небольшая справка, чтобы не тратить время на понимание классов техники: в JSON есть поле “vehicle_type”;
кнопки фильтра на макете соответствуют значениям lightTank, mediumTank, heavyTank, AT-SPG, SPG.
